# gl<sup>3</sup>

WebGL 開発の入門に特化した勉強用ライブラリ。


## 思想

最初にこのライブラリを使って、まず動かすことができるようになる。

つぎに、ライブラリのなかで何が起こっているかを掘り下げてみる。

最終的に自分でスクラッチ WebGL 実装ができるようになることを目指す。

※自分が生産性高めるために使いたいという裏意図がある

## めざすところ

wgld.org で解説していることがほぼ実行できる程度の汎用性。
